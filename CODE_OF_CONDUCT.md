# Code of Conduct

## Purpose

This repository supports the public dissemination of the City4Climate DT Framework. Collaboration should therefore remain respectful, professional, and aligned with the scientific, technical, and public-interest nature of the project.

## Expected behaviour

Contributors and participants are expected to:

- communicate respectfully and constructively;
- focus discussions on the repository scope and technical content;
- provide grounded, transparent, and good-faith feedback;
- respect intellectual contributions, attribution, and licensing conditions;
- act in a way consistent with ethical and scientifically responsible research practice.

## Unacceptable behaviour

The following behaviours are not acceptable in this repository:

- harassment, insults, intimidation, or hostile interactions;
- discriminatory or offensive language;
- deliberate misrepresentation of technical results or project status;
- publication of confidential, restricted, or non-cleared materials;
- sharing of personal data, credentials, secrets, or sensitive operational details.

## Dissemination and confidentiality

Because this is a public repository, contributors must ensure that submitted material is suitable for open dissemination. If the disclosure status of any content is unclear, it should not be published until validated.

## Enforcement

Repository maintainers may edit, hide, lock, or remove content that violates this code of conduct. Repeated or serious breaches may lead to restriction of participation.

## Reporting

Potential issues should be reported to the repository maintainers through the appropriate GitHub channels or through the project coordination contacts when available.

## Scope

This code of conduct applies to repository discussions, issues, pull requests, review comments, and other interactions directly associated with this repository.
