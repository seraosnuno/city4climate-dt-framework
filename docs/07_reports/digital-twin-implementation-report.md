## 1. Introduction

This document is the public Digital Twin Implementation Report associated with the City4Climate project. It describes the implementation-oriented artefacts made available to support reuse of the framework, including source code, documentation, configuration guidance, and demonstration references.

## 2. Objective of the report

This report is intended to document, in a structured and reusable manner:

- the implemented digital twin components made available for public reuse;
- the technical organization of the source code;
- the setup and execution conditions;
- the main dependencies and interfaces;
- the validation status of the public implementation assets.

## 3. Scope

The public implementation scope covered by this report includes only the artefacts approved for open dissemination through this repository.

The report does not assume publication of restricted data, confidential partner materials, or operational deployment details that are outside the approved dissemination perimeter.
