# Open Access Statement

The City4Climate DT Framework repository is organized to support open dissemination of public project outputs, in line with the open and reusable character expected for the digital twin artefacts made available through the project.

Scientific publications and other public results associated with the project should be disseminated in accordance with the applicable open access requirements and the dissemination rules validated by the project coordination.

This repository focuses on the software, documentation, reuse guidance, and technical artefacts that can be publicly released without compromising contractual, legal, ethical, or data protection obligations.

Where necessary, public dissemination may rely on documentation, synthetic data, schemas, and approved demonstrations rather than on operational datasets.
