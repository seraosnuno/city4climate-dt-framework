# Software License Rationale

This repository adopts a permissive software licensing approach to support broad reuse of the public implementation artefacts.

A permissive license is appropriate for a repository intended to maximize technical reuse, adaptation, and dissemination across different contexts.

The selected license applies only to the materials for which the repository owners hold the necessary rights to publish under that license. Third-party materials remain subject to their own licensing conditions and should not be included here unless redistribution is allowed.
