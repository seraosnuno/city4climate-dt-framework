# Dissemination Policy

## Objective

This repository supports the public dissemination of the released first version of the City4Climate Digital Twin framework, within the dissemination perimeter validated for public availability and reuse.

## Dissemination goals

The repository should contribute to:

- public visibility of the project's non-sensitive outcomes;
- technical reuse in other contexts;
- transparent access to documentation and reusable assets;
- support for scientific and technical communication.

## Scope of dissemination

The following classes of outputs are within scope for public dissemination through this repository:

- overview documentation;
- use cases and expected benefits;
- analytical model documentation;
- source code approved for public release;
- demonstration artefacts;
- reuse guidelines;
- release notes and public technical reports.

## Out of scope

This repository is not intended to publish restricted operational information, confidential partner materials, or any content whose disclosure has not been validated.

## Governance

Dissemination decisions should remain aligned with the project coordination, consortium arrangements, and the applicable contractual framework.

## Review

This policy may be updated during the project lifecycle as dissemination needs mature and formal instructions are consolidated.
