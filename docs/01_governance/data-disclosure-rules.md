# Data Disclosure Rules

## Purpose

This document defines the baseline rules for deciding what may be published in the public City4Climate DT Framework repository.

## Core principle

Only materials explicitly cleared for public dissemination may be committed to this repository.

## Materials that may be published

The following categories are generally admissible, subject to validation:

- public technical documentation;
- architectural descriptions;
- methodological documents;
- analytical model descriptions;
- source code approved for open dissemination;
- synthetic data;
- anonymised examples;
- schemas, interfaces, and sample configurations;
- screenshots and demonstration material that do not expose sensitive information.

## Materials that must not be published

The following categories must not be published unless formally cleared:

- personal data;
- confidential or restricted municipal data;
- operational datasets not approved for disclosure;
- credentials, secrets, API keys, tokens, or certificates;
- private endpoints, internal network details, or sensitive infrastructure information;
- partner-confidential materials;
- third-party assets without redistribution rights;
- draft consortium documents not approved for public release.

## Preferred disclosure pattern

Whenever possible, public dissemination should prioritize:

- synthetic data instead of raw operational data;
- anonymised or aggregated examples instead of identifiable records;
- schemas and metadata descriptions instead of live extracts;
- explanatory screenshots instead of direct data exports.

## Validation rule

Before publishing new material, the responsible contributor should confirm that the content is compatible with:

- contractual dissemination conditions;
- consortium-level agreements;
- applicable data protection obligations;
- partner and municipal disclosure constraints.

## Repository note

This repository is intended for open dissemination and reuse. Publication decisions should therefore be conservative by default whenever disclosure status is unclear.
